# Main GrandLisboa Pages

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面的集中存放与版本管理使用。

## 项目简介

本项目旨在提供一个干净、结构化的空间，用于存放各类独立的 HTML 页面文件。这些页面可能是临时演示、实验性内容、资料归档或其他用途的静态页面，均与本仓库自身无关。

## 目录说明

仓库主要目录结构如下（可能根据实际内容调整）：

```
main-grandlisboa-pages/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放各个独立 HTML 页面
│   ├── page1.html
│   └── page2.html
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   └── js/
└── README.md           # 本文件
```

- `pages/`：存放所有归档的独立 HTML 页面文件。
- `assets/`：存放页面可能用到的通用样式、脚本或图片资源。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可以单独访问。
- 页面之间没有依赖关系，除非在页面内部明确引用。
- 归档的页面可能包含实验性、临时性或仅供内部参考的内容。
- 本仓库不保证所有页面长期可用或内容更新。

## 维护说明

- 本仓库由维护者不定期更新，添加或移除页面。
- 欢迎提交 Issue 或 Pull Request 以提出建议或修正。
- 提交新页面时，请遵循以下简单规则：
  - 页面文件放入 `pages/` 目录。
  - 如有公共资源，请放入 `assets/` 对应子目录。
  - 保持页面文件命名简洁、可读。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/yourusername/main-grandlisboa-pages.git
   ```
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件即可查看。
